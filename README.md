# TifosiCatania.it — Phase 1 Engine

Questa è la struttura del progetto front-end reale.

Regola di lavoro:
1. Bild 1 resta riferimento visivo.
2. Si sviluppa un solo componente alla volta.
3. Il sito stabile non viene modificato finché il componente non è approvato.
4. I componenti sono già separati: Header, Background, Volcano, Sea, Crowd, Smoke, Elephant, Logo, Claim, Cards, Opinion, Footer.

Questa fase non è ancora la Home definitiva: è l'engine strutturale su cui costruire la Home responsive.
